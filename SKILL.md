# Slopstopper: Detect and Remove AI Slop

You take a piece of writing, find every tell that it was generated by an AI (ChatGPT, Claude, Gemini, Grok, and the rest), and rewrite it so it reads like a human wrote it. The meaning stays. The voice gets sharper. The slop goes.

This skill is self-contained. Everything you need is in this file plus the bundled `reference/ai-phrase-database.md`. Do not call or depend on any other skill.

---

## Step 0: Get the input

The user points you at text in one of three ways. Handle each:

- **A URL** ("clean up this article: https://..."): fetch the page with WebFetch, asking it to return the full article body text verbatim. Strip nav, ads, cookie banners, and boilerplate. Work on the article prose.
- **A file path** (.md, .txt, .docx-extracted, .html): Read it. For HTML, work on the visible text content.
- **Pasted text** — work on it directly.

If the source is unclear or empty, ask which text to clean before doing anything else.

Always show the user the rewritten text. When the input was a file the user wants edited in place, edit the file; otherwise return the cleaned version in your reply.

## The non-negotiables (apply every time)

1. **No em dashes. Ever.** Replace ( — ) with a comma, a colon, a period and new sentence, parentheses, or just delete it. Same for en dashes used as punctuation. This is the single most important rule.
2. **No "it's not X, it's Y" antithesis.** This is the #1 AI tell and the user's top priority. See the next section.
3. **Straight quotes only.** Convert curly quotes (" " ' ') to straight (" ').
4. **No emojis** as decoration, bullets, or section icons.
5. **Name the actor.** Kill passive voice and subjectless fragments. "It was decided" becomes "Leadership decided."
6. **Keep the meaning.** Never invent facts, sources, names, or numbers to fill a gap. If the original is vague, leave it vague or flag it; do not fabricate specifics.

## Priority 1: The antithesis / negation tic

This is the loudest AI fingerprint, so hunt it first and hardest. The pattern negates a framing ("not X") then swaps in a supposedly deeper one ("it's Y"). Skilled writers use it once for effect. AI sprays it everywhere.

**The fix:** state the point directly. Drop the "not X" setup and keep only the real claim, or rewrite as a plain sentence.

Kill all of these shapes (full list with examples in `reference/ai-phrase-database.md`, Category 1):

- "It's not X, it's Y." / "It's not X. It's Y." / "It's not X — it's Y."
- "This isn't X, it's Y." / "This is not X. This is Y." / "That's not X. That's Y."
- "It's not just X, it's Y." / "Not just X, but Y." / "Not only X, but also Y."
- "X isn't just Y, it's Z." / "It's more than X. It's Y."
- "It's not about X, it's about Y." / "It isn't about X. It's about Y."
- "X doesn't just Y, it Z's." / "It doesn't just X. It Y's."
- "It's less about X and more about Y." / "Less X, more Y." / "It's not so much X as Y."
- "Not because X, but because Y."
- "X is no longer Y. It's Z." / "When you X, it stops being Y and becomes Z."
- "Not X. Not Y. Just Z." / "No X, no Y, just Z."
- Trailing negation tags: "..., no setup needed." / "..., no guessing."

**Detection cue:** an explicit `not` / `n't` / `no`, then a positive pivot (`it's`, `they`, `just`, `but`) within a clause or two. Intensifiers (`just`, `only`, `merely`, `simply`) right after the negation make it certain.

Example:
- Slop: "This isn't just a feature update. It's a complete reimagining of how teams work."
- Fixed: "The update changes how teams share files and assign tasks."

## Priority 2: Strip AI vocabulary and clichés

Replace inflated words with plain ones. Common offenders and swaps:

| Remove | Use instead |
|--------|-------------|
| delve into | look at, examine |
| leverage / utilize | use |
| robust | strong, solid |
| seamless | smooth |
| comprehensive | full, complete |
| holistic | whole |
| navigate (figurative) | handle, deal with |
| underscore / highlight (verb) | show |
| showcase | show |
| foster / cultivate | build, grow |
| testament to | shows, proves |
| pivotal / crucial / vital | important, key (or cut) |
| game changer | (describe what changed) |
| in the realm of / the world of | in |
| unlock the potential of | (say what it does) |
| tapestry / landscape / ecosystem (figurative) | (cut or name the real thing) |

The full word bank (130+ verbs, adjectives, nouns, adverbs and collocations) is in `reference/ai-phrase-database.md`, Category 4. Also kill cliché metaphors: "pave the way," "shed light on," "at the forefront of," "double-edged sword," "tip of the iceberg," "harness the power of" (Category 10).

## Priority 3: Cut throat-clearing, filler, and conclusions

Delete these openers and connectives outright (full lists in the database, Categories 2, 3, 5):

- Openers: "In today's fast-paced world," "In the ever-evolving landscape of," "In the realm of," "Let's dive in," "Imagine a world where," "When it comes to," "Nestled in the heart of."
- Filler: "It's worth noting that," "It's important to note that," "That said," "At the end of the day," "In essence," "At its core," "Needless to say."
- Conclusions: "In conclusion," "Ultimately," "The future looks bright," "The possibilities are endless," "Whether you're a X or a Y," "Only time will tell."

State the point without announcing it first.

---

## Priority 4: Promotional language and chatbot artifacts

- **Promotional inflation** (Category 6): "stands as a testament," "rich tapestry," "plays a vital role," "nestled," "boasts," "breathtaking," "must-visit," "renowned," "state-of-the-art." Replace with a neutral, factual statement.
- **Chatbot artifacts** (Category 7): these are dead giveaways pasted in by mistake. Delete every one: "I hope this helps," "As an AI language model," "Certainly!," "Great question!," "Let me know if," "Feel free to reach out," "As of my last knowledge update," "Based on the information provided."
- **Vague attributions:** "experts believe," "studies show," "observers have noted" without a named source. Name the source or cut the claim.

---

## Priority 5: Structural patterns

- **Rule of three** (Category 8a): AI forces ideas into groups of three. "Robust, scalable, and secure." Break up back-to-back tricolons; keep one or two items where two is enough.
- **"From X to Y" false ranges:** "from startups to Fortune 500s," "whether you're a X, a Y, or a Z." Cut the fake spectrum and name the real thing.
- **Superficial "-ing" tails** (Category 8b): a participle clause bolted on for fake depth: "..., highlighting the importance of X," "..., underscoring the need for Y," "..., paving the way for Z." Delete the tail or fold the real point into the sentence.
- **Formatting tells** (Category 9): excessive bold on key terms, bullet lists where prose belongs, inline-header bullets ("**Speed:** ..."), title-case headings, "Key Takeaways" wrap-up blocks. Flatten into prose; use sentence case.

---

## Detect on density, not single hits

Every phrase here also shows up in human writing. One match is not a verdict. The reliable signal is **clustering** (several tells in one passage) and **density** (the same shape recurring paragraph after paragraph). Weight your judgement: rewrite the obvious tells always; for borderline cases, fix them when the surrounding text is already slop-heavy, and leave genuinely human-sounding lines alone. Do not flatten real voice into bland neutrality. The goal is human writing, not no writing.

---

## Add a human pulse

Removing slop is half the job. Sterile, voiceless text reads as AI too. After cleaning:

- Vary sentence length. Short ones. Then a longer one that takes its time.
- Let opinions and specifics through. Concrete beats vague.
- Use simple verbs (is, are, has) instead of "serves as," "functions as," "stands as."
- Read it aloud in your head. If it sounds like a press release, loosen it.

## Workflow

1. Get the input (Step 0).
2. Scan for the patterns above, Priority 1 first.
3. Rewrite each flagged section, preserving meaning and matching the original tone.
4. Run the non-negotiables check: zero em dashes, zero curly quotes, zero emojis, zero antithesis tics, no fabricated facts.
5. Self-audit. Ask: "What still makes this read as AI?" Answer honestly, then fix what you find.
6. Score it (below). Below 35, revise again.
7. Deliver the cleaned text (or edit the file in place if that was the ask), with a short list of the main changes.

---

## Scoring check (before delivering)

Rate 1-10 each. Total below 35 means revise.

| Dimension | Check |
|-----------|-------|
| Directness | Does each sentence say the thing, or announce it first? |
| Rhythm | Do sentence lengths vary, or all the same cadence? |
| Trust | Are claims specific, or vague and hedged? |
| Authenticity | Sounds like a person, or a model predicting tokens? |
| Density | Is every word earning its place? |

Total /50. Below 35 = revise. 35-42 = acceptable. 43+ = clean.
